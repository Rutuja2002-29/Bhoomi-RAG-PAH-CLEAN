# 🌾 Smart Agri Chatbot (RAG-Based)

An AI-powered Smart Agriculture Chatbot that provides accurate, context-based answers using Retrieval-Augmented Generation (RAG).

This system helps users get reliable information on Rice and Wheat cultivation, fertilizers, soil types, irrigation, and crop management.

---

## 🚀 Features

- 🌾 Supports Rice & Wheat crops
- 📄 Uses real agricultural data (PDF-based knowledge)
- ⚡ Fast semantic search using embeddings
- 🧠 Context-aware answers using RAG
- 🔍 Accurate and reliable responses

---

## 🧰 Tech Stack

- Python  
- ChromaDB (Vector Database)  
- Sentence Transformers  
- Groq API (LLM)  

---

## 📊 Use Cases

- 👨‍🌾 Farmer advisory  
- 🌱 Crop lifecycle guidance  
- 💧 Irrigation planning  
- 🧪 Fertilizer recommendations
